# FoodTrends: Understanding Customer Preferences in FB
# Project Overview
This project focuses on analyzing food habits, nutrition patterns, and lifestyle behavior using Power BI dashboards.
The goal is to understand how dietary preferences, health conditions, and activity levels influence nutritional intake and decision-making.
The dashboards are designed to be interactive, insight-driven, and easy to interpret, helping users make better lifestyle and dietary choices.

# Objectives

-Understand customer food preferences across age groups and demographics

-Analyze nutrition intake (calories, protein, fat, etc.)

-Study the impact of lifestyle and diseases on diet patterns

-Provide data-driven insights to improve health-related decisions

# Dashboard Overview

# 1.Dietary Preferences & Averages
Focus: Understanding eating habits and demographic patterns

# Key Insights:

-Calorie intake remains relatively stable across age groups with slight fluctuations

-Food preferences vary significantly across different age groups

-Lifestyle factors like diseases impact diet adherence and exercise levels

-Gender-based calorie consumption shows minimal variation

# 2.Recipe Nutrition & Preparation Insights
Focus: Ingredient-level nutrition and cooking behavior

# Key Insights:

-Certain ingredients like butter and eggs contribute heavily to cholesterol

-Sugar content varies widely across ingredients, highlighting unhealthy patterns

-Most recipes fall under medium to long preparation time

-Fiber-rich ingredients like lentils and seeds contribute significantly to nutrition

# 3.Averages by Breakfast Suggestions
Focus: Breakfast habits and nutritional impact

# Key Insights:

-Breakfast choices significantly influence daily calorie and sodium intake

-Certain meals provide higher protein but also increased calorie intake

-Activity level and diet type affect daily calorie targets

-Key influencers highlight factors affecting breakfast preferences

# 4.Lifestyle & Nutrition Impact
Focus: Relationship between activity levels and nutrition

# Key Insights:

-Higher activity levels are associated with higher calorie consumption

-BMI tends to decrease with increasing activity level

-Macronutrient distribution varies across diet types

-Sedentary individuals show different nutrition patterns compared to active ones

# Tools & Technologies

Power BI – Data visualization & dashboard creation

Excel / Dataset – Data source

DAX – Calculations and measures

# Design Approach

Used a consistent color theme (orange + neutral tones) for visual harmony

Focused on minimal and clean layout for better readability

Prioritized insightful titles instead of technical labels

Designed dashboards to support quick decision-making

# Key Learnings

Importance of asking the right questions before building dashboards

How to choose the right visuals for storytelling

Balancing design aesthetics with analytical clarity

Understanding real-world application of data visualization

# Future Improvements

Add predictive analytics (calorie recommendations, diet suggestions)

Improve interactivity with advanced filters

Include real-time or larger datasets

Enhance storytelling with more actionable insights

# Conclusion

This project demonstrates how data can be used to analyze food behavior and improve lifestyle decisions.
The dashboards aim to bridge the gap between raw data and meaningful insights, making information accessible and actionable.
